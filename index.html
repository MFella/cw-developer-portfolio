Technologies

  • Angular

    Angular

    One of the top javascript framework. Creating websites becomes easier, and communication between components is clearly visible. The most important pros:

    using Typescript(easier to catch error)

    using RxJS(handling Observables)

    works with custom directives, services, pipes, resolvers

    creating routing with guards

    creating animations

  • Javascript

    Javascript

    That's the one of the easiest and effective languages used to extend functionality in websites. Most important abilities:

    DOM Manipulation - helpful for handling events

    mutations on arrays

    using lambda expression

    making AJAX calls

    integration with Babel and Webpack

    using jQuery

  • SCSS

    SCSS

    Named 'CSS with superpowers'. There is no doubts, that this is true. SCSS have a lot of advantages, for instance:

    creating own variables

    nesting

    separate files(few scss files, and then just use @use)

    mixins - allows make groups of CSS declarations and use it throughout site

    extends - sharing set of CSS properties from one selector to another

    operators - powerful in calculations

  • Karma&Jasmine

    Jasmine + Karma

    That couple makes that, unit testing in Angular becomes easier. Can be used to open a browser, which loads an application's source code and executes tests. Top pros:

    assertions are built into it

    comes with test doubles by using spies

    mocking http controller

    faking activated routes, services, observables, etc.

    behaviour-driven development, and supports test driven development

  • ASP.Net Core

    ASP.Net Core

    Cross-platform framework for building modern cloud based internet connected applications. Powerful tool which is assosiated with this framework and which I used, is Entity Framework. This kind of backend in my projects was used as middleware.

    using c# syntax

    using libraries, such as Linq, AutoMapper, JwtBearer, etc.

    handling request with controllers, communication with database

    using repository pattern

  • xUnit

    xUnit

    Tool, used in unit testing(backend). In my recent projects also heplful in testing controller classes(what functions return). Pros:

    using Moq

    AAA pattern(Arrange-Act-Assert)

Projects

  • Carpart Store

    Web application written in Angular(frontend), ASP.Net Core(middleware), and connected with database(SQLite). The purpose of this application was to imitate an online store with car parts. Normal user can add(or delete) products from basket, order the items, browse the products. Administrator can add products, change the status of orders, etc. Moreover, there are some unit tests(on backend and frontend). More about this project can be found on github.

  • Dating App

    Web application with fully working backend(ASP.Net Core), frontend(Angular), and connected with the database(SQLite). To put it simply, Dating App was created for dating. User can browse and filter other users, write a messages(quasi CRUD system) to another user, add photos, etc. I cordially invite you to visit my gthub for more.

  • Forkify

    The purpose of this application was to create recipe ebook. User can search dishes, check the ingredients, and so on. Project in large part is written in Javascript. Beside that, webpack and babel (polyfill) are used here. More info on github.
